# dcgan_with_encoder
Implementation of DC-GAN with Encoder using TensorLayer 2.0

## Requirement
- Python3
- TensorFlow==2.0.0a0
- TensorLayer==2.1.0
- CPU or NVIDIA GPU + CUDA CuDNN

## Train the Model

Clone this repo

```python
python3 train.py
```
